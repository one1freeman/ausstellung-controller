# Haupt-Mikrocontroller für eine Ausstellung des GMI

Dieser Mikrocontroller mit seinen angeschlossenen Modulen erfüllt mehrere Aufgaben in der Steuerung der Ausstellung:
- Darstellung und Aufnahme (durch Annahme der Daten des [Temperatursensors](https://github.com/one1freeman/ausstellung-temperatur-sensor) sowie eigener Messung der "Aussentemperatur") der Temperaturdaten auf dem OLED Panel
- Hosten eines Access Points sowie eines Webservers für die Einsicht des Status der Ausstellung sowie das Ändern des Modus
<br><img width="700" alt="grafik" src="https://github.com/user-attachments/assets/0c344f3b-46c0-4c32-b3aa-a9381c052f98" /><br>
- Zeitschaltung der mit den Modulen (Lüfter für "Windenergie", Lampe für "Solarenergie" und Wärmepumpe) verbundenen Relais
